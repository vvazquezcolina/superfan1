Brand Asset Scraper CLI

Web Version

Powerful CLI Tool for Developers

Get the full power of Brand Asset Scraper with our command-line interface

Installation

1

Clone the Repository

git clone https://github.com/vvazquezcolina/web-scrapper.git
cd web-scrapper
2

Install Dependencies

pip install -r requirements.txt
3

Run the Scraper

python main.py https://example.com

Usage Examples

Basic Usage

python main.py https://stripe.com

Custom Output Directory

python main.py https://airbnb.com -o ./airbnb_assets

Advanced Options

python main.py https://apple.com -d 2 --max-pages 50 --verbose

Skip Video Processing

python main.py https://nike.com --skip-videos --no-optimize

Command Line Options

Option Description Default
-o, --output Output directory path ./scraped_assets
-d, --depth Maximum crawl depth 3
--max-pages Maximum pages to scrape 100
--skip-videos Skip video extraction False
--no-optimize Skip image optimization False
-v, --verbose Enable detailed logging False
