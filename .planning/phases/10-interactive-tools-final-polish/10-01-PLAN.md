**Step 1: Create src/components/tools/InteractiveMap.tsx**

This is a `'use client'` component. Leaflet must be dynamically imported to avoid SSR errors (`window is not defined`). Use Next.js dynamic import with `ssr: false`.

```typescript
'use client'

```

BUT — Leaflet cannot be used with the standard `dynamic()` approach easily because its API is imperative. The correct pattern for Leaflet in Next.js App Router is:

```typescript
// Do NOT import L from 'leaflet' at the top level.
// Use useEffect to dynamically import and initialize the map.

'use client'
import { useEffect, useRef } from 'react'
import { trackToolUsage } from '@/lib/analytics'

export interface StadiumMarker {
  id: string
  name: string      // already localized
  cityId: string
  cityName: string  // already localized
  capacity: number
  lat: number
  lng: number
  countryColor: string  // 'green' | 'blue' | 'red'
  stadiumSlug: string   // for /es/estadios/{slug}
  citySlug: string      // for /es/ciudades/{slug}
}

interface InteractiveMapProps {
  stadiums: StadiumMarker[]
  lang: 'es' | 'en'
  dict: {
    mapLoading: string
    popupCapacity: string
    popupCityGuide: string
    popupStadiumPage: string
  }
}
```

Inside the component, use `useEffect` to:
1. Import Leaflet dynamically: `const L = (await import('leaflet')).default`
2. Import Leaflet CSS: dynamic CSS injection via `import('leaflet/dist/leaflet.css')` — but CSS imports inside useEffect are not standard. Instead, add a `<link>` tag to the document head inside the effect, or use a `<style>` tag. The cleanest approach: in `useEffect`, create a `<link rel="stylesheet">` element and append to document head if not already present.
3. Initialize map centered at `[38.5, -98]` (center of North America) with zoom level 4
4. Add OpenStreetMap tile layer: `https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png`
5. For each stadium in `stadiums` prop:
   - Create a circle marker (not default pin) using `L.circleMarker([lat, lng], { radius: 10, fillColor: countryColor, color: '#fff', weight: 2, opacity: 1, fillOpacity: 0.9 })`
   - Bind popup HTML string:
     ```html
     <div class="leaflet-popup-content-inner">
       <h3 style="font-weight:bold;margin-bottom:4px">${name}</h3>
       <p style="margin:0 0 4px 0">${cityName}</p>
       <p style="margin:0 0 8px 0">${dict.popupCapacity}: ${capacity.toLocaleString()}</p>
       <a href="/${lang}/ciudades/${citySlug}" style="display:block;margin-bottom:4px;color:#16a34a">${dict.popupCityGuide}</a>
       <a href="/${lang}/estadios/${stadiumSlug}" style="color:#16a34a">${dict.popupStadiumPage}</a>
     </div>
     ```
   - On marker click: call `trackToolUsage('interactive_map')` (per D-10). Use `marker.on('click', () => trackToolUsage('interactive_map'))`
6. Return cleanup: `map.remove()` in useEffect cleanup function

Map container div needs explicit height: `<div ref={mapRef} style={{ height: '500px', width: '100%' }} className="rounded-lg overflow-hidden" />`
